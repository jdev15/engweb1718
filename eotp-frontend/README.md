# Como usar
## Configuração
Executa o seguinte comando na linha de comandos para instalar os pacotes necessários para correr a _frontend_:
```
npm install
```
Para iniciar o projeto, simplesmente corre:
```
npm start
```

# Componentes existentes
* Sidebar
* Navbar
* Tabela comum
* Página Registo
